Sobre o Projeto

Este projeto web tem como objetivo apresentar o jogo de dedução e lógica Black Box de uma forma visualmente atraente e informativa. A página foi estruturada para ser um ponto de partida para a criação de uma versão jogável do Black Box, com um foco especial na separação de conteúdo (HTML) e estilo (CSS).

O design minimalista e as cores vibrantes buscam refletir a natureza do jogo, que combina lógica e uma estética tecnológica de "caixa preta".

Objetivos

  • Criar uma estrutura de página web robusta e semântica utilizando HTML5.
  • Aplicar técnicas de estilização com CSS, explorando seletores e propriedades.
  • Compreender o conceito de links internos e navegação entre páginas.
  • Organizar o projeto de forma modular e clara, pensando na futura implementação de funcionalidades.

Disciplina

O desenvolvimento deste projeto faz parte da disciplina de Programação Web. A proposta visa consolidar conhecimentos sobre a tríade fundamental da web: HTML para a estrutura, CSS para o estilo e, em um futuro próximo, JavaScript para a interatividade.

O foco inicial é garantir uma base sólida de HTML e CSS, que são a fundação de qualquer aplicação web.
