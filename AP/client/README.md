# Black Box — Bot & 1v1 (rodadas)

## Como rodar localmente
1. Extraia o projeto.
2. No terminal, entre na pasta e rode:
   ```bash
   npm install
   npm run dev
   ```
3. Abra o navegador em http://localhost:5173

## Como publicar
- Rode `npm run build`. Os arquivos finais ficarão em `dist/`.
- Publique a pasta `dist/` no Netlify, Vercel, GitHub Pages, etc.
